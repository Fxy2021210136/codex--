# 更新日志

## v0.1.0 - 公开试用基线

这是当前项目的第一个 GitHub 版本管理基线，适合本机演示、内测和继续准备公开部署。

### 已包含

- 通用工程计划工作台：横道图、网络图、WBS、工序库、资源负荷、计划审查、关键节点和汇报版。
- 多工程类型模板：房建、道路、桥梁、市政管网、工业安装和自定义工程。
- 自定义模板：支持阶段、专业、工期、前置关系、FS/SS/FF/SF、Lag、资源、材料节点和空间维度。
- 后端持久化：SQLite 保存项目、账号、Session、自定义模板和 AI 配置。
- 账号体系：邮箱密码注册登录、用户项目隔离、密码修改、账号数据导出和账号删除。
- 管理员能力：管理员邮箱、项目配额、AI 每日额度、运营概览和上线健康检查。
- AI 接入：DeepSeek、Gemini、OpenAI Responses 服务端代理，API Key 不进入浏览器。
- Codex 智能体桥接：管理员专用，默认关闭，默认只读沙箱。
- 部署准备：Docker、GitHub Actions、生产环境变量模板、公开部署清单、本机备份/恢复脚本。

### 适合的使用方式

- 本机运行 + SQLite 保存数据。
- Cloudflare Tunnel 临时给别人试用。
- 迁移到支持持久磁盘的平台做长期部署。

### 暂不建议

- 直接用 GitHub Pages 承载完整功能，因为后端 API 无法在 Pages 上运行。
- 没有持久磁盘就开放多人长期保存数据。
- 在公共匿名站点开放 Codex 写入权限。
